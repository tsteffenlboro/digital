# Digital Engineering

This is a trial site for the module Digital Engineering. It uses JupyterLite on GitHub Pages to provide an interactive experience. 

https://tsteffenlboro.github.io/digital/

## Requirements

JupyterLite is being tested against modern web browsers:

- Firefox 90+
- Chromium 89+

It does seem to work fine in Google Chrome, Microsoft Edge, and Apple Safari, too.
